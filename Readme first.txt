English : (Türkçe için aşağıya bakınız.)

This project using jsonplaceholder. You must do these before starting npm , enter directory from terminal after that write this code :

npx json-server --watch src/api/movies.json --port 5000

After doing this, you can start npm or yarn.

Türkçe :

Bu proje jsonplaceholder kullanıyor. Bu yüzden npm'i başlatmadan önce terminalden projenin dizinine girin ardından şu kodu yazın :

npx json-server --watch src/api/movies.json --port 5000

Bunu yaptıktan sonra npm'i veya yarn'ı başlatabilirsiniz.
